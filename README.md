## 🎬 SHITAI 画面イメージ集

![alt text](image/shitai.drawio.png)

このドキュメントでは、サービス「SHITAI」の各画面イメージを紹介します。<br>
実際の UI を確認することで、利用者が操作の流れを直感的に理解できるようにしています。

### 🏠 ホーム画面

グループ内の「したいこと」を一覧で確認できます。<br>
新着順・期限接近順などのタブで切り替えが可能です。

![alt text](image/ホーム画面.png)

### 🏆 ランキング画面

未確定の「したいこと」を参加人数順に表示します。<br>
もっとも人気のある企画が一目で分かります。

![alt text](image/ランキング画面.png)

### ✏️ したいこと作成画面

ユーザーが新しい「したいこと」を投稿するための画面です。<br>
タイトル、カテゴリー、画像、期限、参加確認項目などを設定できます。<br>

| 画面名                                     | 説明                                     |
| ------------------------------------------ | ---------------------------------------- |
| ![alt text](image/したいこと作成画面.png)  | 基本情報（タイトル、カテゴリ、画像）入力 |
| ![alt text](image/したいこと作成画面2.png) | 参加条件・期限・人数設定                 |
| ![alt text](image/したいこと作成画面3.png) | 参加時／確定時の確認項目設定             |

### 🔍 したいこと詳細画面

投稿された「したいこと」の詳細を確認できます。<br>
現在の参加人数や締切、備考などを表示し、「参加する」ボタンから参加可能です。<br>

![alt text](image/したいこと詳細画面.png)
![alt text](image/したいこと詳細画面2.png)

### 🔍 各ユーザーごとのしたいこと参加状況確認画面

どのユーザーがいつどのしたいことに参加しているか確認することができます。<br>
参加状況を確認することにより、したいことの実施日を決めやすくなります。<br>

![alt text](image/したいこと参加状況確認.png)
![alt text](image/したいこと参加状況確認2.png)

![alt text](image/参加時の確認入力画面.png)
